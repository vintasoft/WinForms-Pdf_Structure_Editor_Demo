The project contains the following compiler directives:
1. REMOVE_JBIG2_PLUGIN - allows to disable code, which references to VintaSoft JBIG2 .NET Plug-in
2. REMOVE_JPEG2000_PLUGIN - allows to disable code, which references to VintaSoft JPEG2000 .NET Plug-in
3. REMOVE_DOCCLEANUP_PLUGIN - allows to disable code, which references to VintaSoft Document Cleanup .NET Plug-in
